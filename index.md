# tree-sitter-goluca

Tree-sitter grammar for [Go-Luca](https://codeberg.org/hum3/go-luca) accounting files (`.goluca`).

Provides parsing, syntax highlighting, and editor integration for the Go-Luca
double-entry transaction format.
